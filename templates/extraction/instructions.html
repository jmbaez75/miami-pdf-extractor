1. Single map generation

Creates the reference map required for extraction.

  • Setup: Provide the path to a sample PDF that represents the layout of your documents.
  • Execution: Click "Execute Mapping". This scans the PDF layout and generates the mapa_resultado.csv file.
  • You can store differentet maps for each kind of document.

2. PDF Batch Processing

The control center for processing your masive batch of PDF files.

  • Paths: Set the absolute paths for your CSV map, input PDFs, and Excel output.
  • Sensitivity: Adjust the slider (px tolerance) if the extractor struggles to match data due to layout variations.
  • Execution: Click "Execute Batch" to process all PDFs in your folder based on your configuration.

3. Output Columns (Output columns)

Defines the structure of your final Excel report.

  • Mapping: After running a map in Map Execution, visit this tab to see all detected text.
  • Configuration: Type your desired header name in the "Nombre columna" box for the data you want to export. Leave fields blank to ignore them.
  • Save: Click "Guardar etiquetas" to commit your column choices to the map file.

4. Filters (Filters)

Automates data cleaning during extraction.

  • Purpose: Define rules to replace specific text strings (e.g., removing symbols or correcting labels).
  • Usage: Add a row, input the "Expresión a Buscar" (target), and define the "Reemplazo" (replacement).
  • Save: Click "Guardar Filtros" to apply these cleaning rules to future batches.
